# PROPOSED — scope-creep catcher

> **Hard rule for all agents:** you may implement **only** tasks already in [sprint-state.json](sprint-state.json) `tasks[]`. If you notice something worth doing that is not a backlog task — a refactor, a new feature, a nice-to-have, a dependency upgrade — you append it here and move on. You do **not** build it. The founder triages this list and promotes items into the backlog (or rejects them).

This is how "lean and mean, no scope creep" is enforced against an autonomous fleet: the backlog is the only source of work, and this file is the pressure-release valve so good ideas are captured without derailing the sprint.

## Format
`- [PROP-n] (from T-xxx, tick N) — <idea> · rationale · rough size`
